# IPLセレクタ改 for MZ-80K/C,1200

## はじめに
これはHideki Suga(@junk_suga)さんが作成したMZ-80K/C,1200のIPLセレクタをkuran_kuranが改良(?)したものです。
私が手を入れた事によって予期せぬバグも一緒に入った可能性があります。

オリジナルはこちらです  
https://web.archive.org/web/20230123144622/https://www.ne.jp/asahi/suga/junkyard/mz/mzsoft/iplselkc/index.html

## 配布内容
```
　IPLSEL03.D88        : IPLセレクター ディスクイメージ(「IPLS MAKER K/C」、「TAKEOUT(FD2CMT)K」入り)
　MKIPLSKC.mzt        : IPLセレクターメーカー テープイメージ(mzt)
　MKIPLSKC.WAV        : IPLセレクターメーカー テープイメージ(wav)
　IPLSKC.ASM          : IPLセレクター ソース
　MKIPLSKC.ASM        : IPLセレクターメーカー ソース
  MAKE.BAT            : コンパイル用バッチファイル(参考)
  ReadMe.txt          : 元のreadme
```
※アセンブラは、紅茶羊羹(@youkan700)さんのZ80ASを使っています。
